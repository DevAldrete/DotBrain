# Dotbrain

High-Performance Workflow Orchestration: Code-First, Visual-Always.

Dotbrain is a distributed workflow engine built in Go, designed to bridge the gap between low-level control and high-level automation. It enables developers to build, visualize, and deploy complex logic pipelines—from simple API bridges to sophisticated AI agentic workflows—with absolute privacy and type safety.

## 🛠️ The Stack

Core Engine: Written in Go for high-concurrency and low-latency execution.

Orchestration: Built for Kubernetes to ensure horizontal scalability and fault tolerance.

AI Strategy:

- Current: Unified interface for external LLM APIs (OpenAI, Anthropic, etc.).

- In the Future: Python-based microservices for specialized ML tasks.

## 🚀 Key Features

- Workflow-as-Code: Define your logic using our Go/TypeScript SDKs. Version control your automation just like your application code.

- Visual Designer: A drag-and-drop interface that syncs bidirectionally with your code.

- Type-Safe Validation: Integrated "Safe Object" nodes for schema validation and data sanitization.

- Agentic AI Nodes: Out-of-the-box support for RAG (Retrieval-Augmented Generation) and tool-calling patterns.
